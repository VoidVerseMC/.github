FAQ

Which version is VoidVerse supported on?
We support versions starting from 1.8.9 to the latest version. Some versions may have bugs. The server is currently running on 1.21.8.
What is VoidVerse for?
VoidVerse is focused on PvP and BedWars and client testing:
• Aqua Client
• Jet Client
• Apple Client
