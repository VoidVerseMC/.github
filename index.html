FAQ

Which versions are supported?
We support versions from 1.8.9 to the latest. Some features may vary per version. Recommended: 1.21.8 for best experience across all servers.
What is the Lifesteal server about?
The Lifesteal server features intense PvP where you steal hearts from opponents. Each kill gives you extra hearts, making you stronger. Die too much and you'll be eliminated! Perfect for hardcore PvP players.
Which clients are supported for testing?
All servers support client testing for:
• Aqua Client
• Jet Client
• Apple Client
